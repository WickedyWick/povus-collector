Okay, So What the Heck?

This is basically copy of already existing surveying web tool with a tiny portion of the code altered

Left click anywhere on the map and pin will be created with 180 second timer (after harvesting first time)

Then click on the timer of the bush you harvested after every harvest.

After 3rd harvest (click) pin will dissapear as bush will too.

When timer expires pins change colors.

If you need to force delete pin, click on it two times.

Currently pin system is functional, but resizing does not work atm

Resizing after posting pins will mess up pins positions

If you want to resize please do before starting to put down pins
